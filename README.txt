Simple form & table app on Angular.js v1.0	
https://github.com/MaxDunaevsky/simpleFormTableAppOnAngularJs.git
(c) 2016 MaxDunaevsky (MaxDunaievskyi@gmail.com)

Features:
- live update of table from form
- table is sortable by columns
- remove option for table records
- input field validation with highlighting
- preventive check for dublicate entries (regarding phone numbers)
- bootstrap style
- no third parties' JS libraries used (except Angular.js itself)
